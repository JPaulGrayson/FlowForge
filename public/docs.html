Quick Start

Get Orchestrate connected to Claude Desktop or Cowork in under a minute.

1. Add to MCP Configuration

Add Orchestrate to your MCP servers configuration:

{
  "mcpServers": {
    "orchestrate": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://orchestrate.us.com/api/mcp/sse"]
    }
  }
}

2. Start Using Orchestrate

In Claude, you can now say things like:

  • "Generate a workflow for processing customer feedback"
  • "Visualize this workflow in LogicArt"
  • "Ask the AI council about best practices for API design"
  • "Save this workflow for later"

Claude Desktop / Cowork Setup

For Claude Desktop

  1. Open Claude Desktop settings
  2. Navigate to MCP Servers configuration
  3. Add the Orchestrate server configuration (see Quick Start)
  4. Restart Claude Desktop

For Cowork

Cowork uses the same MCP protocol. Add Orchestrate to your .mcp.json file:

{
  "mcpServers": {
    "orchestrate": {
      "type": "sse",
      "url": "https://orchestrate.us.com/api/mcp/sse"
    }
  }
}

Available Tools

generate_workflow

Generate a workflow from natural language description.

{
  "name": "generate_workflow",
  "arguments": {
    "prompt": "Create a workflow that searches for news and summarizes it"
  }
}

visualize_workflow

Get a LogicArt visualization URL for a workflow.

{
  "name": "visualize_workflow",
  "arguments": {
    "workflow": { ... }
  }
}

üß† council_query

Query multiple AI models (Claude, GPT-4, Gemini) for consensus. Get unified answers backed by collective intelligence.

Consensus Patterns
consensus Synthesize unified answer from all models
vote Majority wins for discrete choices
compare Side-by-side analysis
debate Models critique each other
Example
{
  "name": "council_query",
  "arguments": {
    "query": "What are the best practices for REST API design?",
    "models": ["claude", "gpt4", "gemini"],
    "pattern": "consensus"
  }
}

Workflow Schema

Workflows follow this TypeScript structure:

interface Workflow {
  id: string;
  name: string;
  description: string;
  version: string;
  config: {
    timeout?: number;
    maxParallelism?: number;
    continueOnError?: boolean;
  };
  nodes: Array<{
    id: string;
    type: 'start' | 'end' | 'tool' | 'decision' | 
          'loop' | 'parallel' | 'join' | 'human' | 'council';
    label: string;
    description?: string;
    config: any;
  }>;
  edges: Array<{
    id: string;
    sourceNodeId: string;
    targetNodeId: string;
    label?: string;
    condition?: object;
  }>;
  startNodeId: string;
  metadata: {
    author?: string;
    createdAt: string;
    updatedAt: string;
    tags?: string[];
    visibility: 'private' | 'team' | 'public';
  };
}

Node Types

Type Description
start Entry point of the workflow
end Exit point of the workflow
tool Executes a tool (web_search, summarize, etc.)
decision Conditional branching based on data
loop Iterate over items
parallel Split into parallel branches
join Merge parallel branches
human Wait for human input
council Query AI council for consensus
