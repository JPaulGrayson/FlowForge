Quick Start

Connect via Claude Desktop or Cowork

Add Orchestrate to your MCP configuration:

{
  "mcpServers": {
    "orchestrate": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://orchestrate.us.com/api/mcp/sse"]
    }
  }
}
